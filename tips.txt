NVIM ALIASES
v - open nvim
n - browse current directory with oil.nvim
vs - fuzzy find files with bat preview
vg - grep file contents and jump to matching lines
o - instantly access Obsidian vault (sorted by recency)
r - recent files across ALL ~/code projects
- (in nvim) - toggle oil.nvim filesystem browser
In oil.nvim: delete a line to delete a file, edit a line to rename it
Oil.nvim trick: visual select multiple files, :d to delete them all

TMUX KEYBINDINGS
C-a g - lazygit popup (floats over panes)
C-a K - yazi file manager popup
C-a S - scratch terminal toggle (persistent)
C-a C-y - yank entire pane scrollback to clipboard
C-a M-y - yank last 200 lines to clipboard
C-a Space - tmux-thumbs (vimium-style hints for copying text)
C-a - - split pane vertically (current path)
C-a _ - split pane horizontally (current path)
C-a > - swap pane down
C-a < - swap pane up
C-a H/J/K/L - resize pane left/down/up/right
C-a h/j/k/l - select pane left/down/up/right
M-1 through M-9 - select pane by number
M-h/M-l - previous/next window
C-a Z - zen mode toggle
C-h/j/k/l - seamless navigation between nvim splits AND tmux panes

NVIM NAVIGATION
gd - goto definition
gr - goto references
gI - goto implementation
gy - goto type definition
Telescope: C-_ opens file in vertical split
<leader>db - toggle breakpoint
<leader>dc - start/continue debugging
<leader>du - toggle debug UI
In .http files: Enter to execute request

MERGE CONFLICTS
co - choose ours (current branch)
ct - choose theirs (incoming)
cb - choose both
c0 - choose none
[x / ]x - jump between conflicts

CLIPBOARD & JSON

PYTHON (UV)
